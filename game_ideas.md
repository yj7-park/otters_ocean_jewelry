# 🎮 게임 아이디어 제안: "Budgie's Bug Shop" 스타일의 힐링 & 성장 게임

"Budgie's Bug Shop"의 핵심 매력인 **아기자기한 캐릭터, 단순하지만 중독성 있는 루프, 수집과 성장의 재미**를 계승한 4가지 게임 아이디어를 제안합니다.

---

## 1. ♨️ 카피바라의 온천 배달 (Capybara's Hot Spring Delivery)
**"느긋한 카피바라와 함께하는 숲속 온천 경영"**

- **컨셉**: 숲속의 관리자인 카피바라가 되어, 다양한 동물 손님들이 원하는 '온천 부재료(과일, 약초, 특수 비누)'를 채집하거나 재배하여 배달합니다.
- **핵심 루프**: 숲 탐험(채집/미니게임) -> 손님 주문 확인 -> 온천탕에 재료 넣기 -> '휴식 포인트' 획득 -> 새로운 탕과 시설 업그레이드.
- **전략/육성**: 특정 동물 조합이 같은 탕에 들어갈 때 발생하는 시너지(예: 원숭이와 코끼리가 같이 있으면 보너스!)와 각 동물의 선호도 관리.
- **스토리**: 잠이 너무 많아 깨어나지 않는 '거대 용'을 기분 좋게 깨우기 위해 최고의 온천을 만드는 여정.

## 2. 💎 두더지의 보석 빵집 (Mole's Crystal Bakery)
**"반짝이는 광물을 달콤한 디저트로!"**

- **컨셉**: 지하 세계의 미식가 두더지가 되어, 땅속에서 캐낸 반짝이는 광물(설탕 결정 같은 보석)들로 빵을 굽습니다.
- **핵심 루프**: 광산 탐험(클리커/타이밍 미니게임) -> 광물 정제 -> 보석 빵 굽기 -> 지하 생물들에게 판매 -> 베이커리 인테리어 확장.
- **전략/육성**: 어떤 광물을 섞느냐에 따라 빵의 효능(빛나는 빵, 몸이 가벼워지는 빵 등)이 달라지며, 이를 통해 단골 손님의 호감도를 올립니다.
- **스토리**: 지상의 해가 사라진 세상, 가장 밝게 빛나는 '태양 빵'을 구워 지상을 다시 밝히려는 따뜻한 이야기.

## 3. 🐚 수달의 바다 보석상 (Otter's Ocean Jewelry)
**"물방울과 조개껍데기로 만드는 바다의 예술"**

- **컨셉**: 패션 감각이 뛰어난 해달(Otter)이 되어, 바닷속 예쁜 조개와 진주를 모아 물고기 손님들에게 장신구를 만들어줍니다.
- **핵심 루프**: 잠수하여 재료 수집(드래그 & 드롭) -> 장신구 제작 -> 패션쇼/가게 판매 -> 진주 머니 획득 -> 잠수 장비 및 제작 도구 강화.
- **전략/육성**: '바다의 유행' 시스템. 시즌마다 유행하는 색상이나 재질이 바뀌어 시장 흐름을 읽는 재미를 줍니다.
- **스토리**: 전설의 거대 고래가 참가하는 '바다 무도회'의 메인 디자이너로 선택받기 위한 도전기.

## 4. ✉️ 나무늘보의 느릿느릿 우체국 (Sloth's Slow Post Office)
**"느리지만 정확하게, 마음을 전달하는 풍선 우편"**

- **컨셉**: 모두가 너무 바쁘게 사는 공중 도시에서, 가장 느긋한 나무늘보가 '풍선 우편물'을 분류하고 배달합니다.
- **핵심 루프**: 우편물 분류(리듬/퍼즐 요소) -> 적절한 풍선 달기 -> 새 친구들을 통한 배달 관리 -> 우체국 꾸미기 및 우체부 새 고용.
- **전략/육성**: 새들의 지구력을 관리하고, 우편물의 무게에 따른 풍선 조합을 최적화하는 전략적 재미.
- **스토리**: 기계화된 빠른 우편 때문에 잊혀진 '손편지의 진심'을 되찾아주는 잔잔한 감동 스토리.
